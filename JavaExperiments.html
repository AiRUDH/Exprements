Experiment - 1
Aim:
Write a program to find the sum of a number of integers entered as command line arguments.
Algorithm:
1. Accept command line arguments passed to the program
2. Convert each argument from String to integer
3. Initialize a variable 'sum' to zero
4. Iterate through each argument:
a. Convert the current argument to integer
b. Add the integer value to the running sum
5. Display the final sum
6. Handle possible NumberFormatException if non-integer arguments are provided
Code:
public class MainCode { public static void main(String[] args) { int sum = 0; if (args.length == 0) { System.out.println("No arguments provided. Please provide integer values as command line arguments."); return; } try { for (String arg : args) { sum += Integer.parseInt(arg); } System.out.println("Sum of the provided integers is: " + sum); } catch (NumberFormatException e) { System.out.println("Error: Please provide valid integer values as command line arguments."); } } }

Experiment - 2
Aim:
Write a program to find the factorial of a number.
Algorithm:
1. Import necessary packages
2. Define a class named MainCode
3. Create a main method
4. Define a method to calculate factorial:
a. If number is 0 or 1, return 1
b. Otherwise, return number * factorial(number - 1)
5. Take input from the user
6. Calculate factorial of the input number
7. Display the result
Code:
import java.util.Scanner; public class MainCode { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Enter a number to calculate factorial: "); int number = scanner.nextInt(); if (number < 0) { System.out.println("Factorial is not defined for negative numbers."); } else { long result = factorial(number); System.out.println("Factorial of " + number + " is: " + result); } scanner.close(); } // Method to calculate factorial public static long factorial(int n) { if (n == 0 || n == 1) { return 1; } else { return n * factorial(n - 1); } } }

import java.io.BufferedReader; import java.io.File; import java.io.FileReader; import java.io.IOException; import java.util.Scanner; public class MainCode { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Enter the file path: "); String filePath = scanner.nextLine(); System.out.print("Enter the string to search for: "); String searchString = scanner.nextLine(); File file = new File(filePath); if (!file.exists()) { System.out.println("File does not exist at the specified path."); scanner.close(); return; } int lineNumber = 0; int occurrences = 0; try (BufferedReader reader = new BufferedReader(new FileReader(file))) { String line; System.out.println("\nSearching for '" + searchString + "' in file: " + filePath); System.out.println("-----------------------------------------------"); while ((line = reader.readLine()) != null) { lineNumber++; if (line.contains(searchString)) { occurrences++; System.out.println("Line " + lineNumber + ": " + line); } } System.out.println("-----------------------------------------------"); if (occurrences > 0) { System.out.println("Found " + occurrences + " occurrence(s) of '" + searchString + "' in the file."); } else { System.out.println("The string '" + searchString + "' was not found in the file."); } } catch (IOException e) { System.out.println("Error reading the file: " + e.getMessage()); } scanner.close(); } }

import java.io.BufferedReader; import java.io.File; import java.io.FileReader; import java.io.IOException; import java.util.Scanner; public class MainCode { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Enter the file path: "); String filePath = scanner.nextLine(); File file = new File(filePath); if (!file.exists()) { System.out.println("File does not exist at the specified path."); scanner.close(); return; } int lineNumber = 0; int commentLines = 0; // Using try-with-resources to automatically close resources try (BufferedReader reader = new BufferedReader(new FileReader(file))) { String line; System.out.println("\nLines starting with '//' in file: " + filePath); System.out.println("-----------------------------------------------"); while ((line = reader.readLine()) != null) { lineNumber++; if (line.trim().startsWith("//")) { commentLines++; System.out.println("Line " + lineNumber + ": " + line); } } System.out.println("-----------------------------------------------"); if (commentLines > 0) { System.out.println("Found " + commentLines + " comment line(s) in the file."); } else { System.out.println("No comment lines found in the file."); } } catch (IOException e) { System.out.println("Error reading the file: " + e.getMessage()); } scanner.close(); } }

Experiment - 6
Aim:
Write a program to check if a number is prime or not, by taking the number as input from the keyboard.
Algorithm:
1. Import necessary packages
2. Define a class named MainCode
3. Create a main method
4. Take input from the user for the number to check
5. If the number is less than or equal to 1, it's not prime
6. If the number is 2, it's prime
7. Check if the number is divisible by any number from 2 to square root of the number
8. If divisible by any number, it's not prime
9. Otherwise, it's prime
10. Display the result
Code:
import java.util.Scanner; public class MainCode { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Enter a number to check if it's prime: "); int number = scanner.nextInt(); boolean isPrime = checkPrime(number); if (isPrime) { System.out.println(number + " is a prime number."); } else { System.out.println(number + " is not a prime number."); } scanner.close(); } public static boolean checkPrime(int n) { if (n <= 1) { return false; } if (n == 2) { return true; } for (int i = 2; i <= Math.sqrt(n); i++) { if (n % i == 0) { return false; } } return true; } }
Output:

Write a program to find the sum of a number of integers (maximum 10), entering every number from the keyboard.
Algorithm:
1. Import necessary packages
2. Define a class named MainCode
3. Create a main method
4. Create a Scanner object for input
5. Ask user for the number of integers (max 10)
6. Use a loop to input numbers
7. Calculate sum while inputting
8. Display the final sum

Code:
import java.util.Scanner; public class MainCode { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); int sum = 0; int count; do { System.out.print("Enter the number of integers (max 10): "); count = scanner.nextInt(); if (count > 10) { System.out.println("Please enter a number less than or equal to 10."); } } while (count > 10); for (int i = 1; i <= count; i++) { System.out.print("Enter number " + i + ": "); sum += scanner.nextInt(); } System.out.println("Sum of the entered numbers is: " + sum); scanner.close(); } }

Write a program to create a class 'Distance' with methods where distance is computed in terms of feet and inches.
Algorithm:

Code:
public class Distance { private int feet; private double inches; // Constructor public Distance(int feet, double inches) { this.feet = feet; this.inches = inches; normalize(); } // Normalize the distance (12 inches = 1 foot) private void normalize() { if (inches >= 12.0) { feet += (int)(inches / 12); inches = inches % 12; } } // Add two distances public Distance add(Distance d) { int newFeet = this.feet + d.feet; double newInches = this.inches + d.inches; return new Distance(newFeet, newInches); } // Convert to string representation public String toString() { return feet + " feet " + inches + " inches"; } public static void main(String[] args) { Distance d1 = new Distance(5, 6.5); Distance d2 = new Distance(3, 8.5); Distance sum = d1.add(d2); System.out.println("Distance 1: " + d1); System.out.println("Distance 2: " + d2); System.out.println("Sum: " + sum); } }
